# Starting Town ASCII Tilemap

- **Size:** 50 columns x 54 rows (tiles).
- **Origin:** top-left is (0,0). Rows increase downward, columns increase rightward.

## Legend

- `.` Grass
- `P` Plaza tiles
- Path tiles (tan dirt path pieces from `Grass_Tiles_1.png`):
  - `A` Horizontal top-left
  - `D` Horizontal top
  - `E` Horizontal top-right
  - `H` Horizontal bottom-left
  - `J` Horizontal bottom
  - `K` Horizontal bottom-right
  - `L` Vertical left
  - `R` Center (pure path)
  - `Q` Vertical right
  - `U` Inner corner top-left
  - `V` Inner corner top-right
  - `X` Inner corner bottom-left
  - `Z` Inner corner bottom-right
- `I` Inn
- `B` Blacksmith
- `S` Market stalls
- `1` House 1 (northeast)
- `2` House 2 (northeast)
- `3` House 3 (west)
- `4` House 4 (southwest)
- `5` House 5 (northwest)
- `F` Fountain
- `N` Bench
- `W` Well
- `b` Barrel
- `h` Hay bale
- `f` Fence
- `p` Flower pot
- `g` Sign
- `T` NPC (Bartender)
- `M` NPC (Miner)
- `C` NPC (Chef)
- `O` Tree (Oak)
- `r` Tree (Birch)
- `Y` Tree (Spruce)
